Description

This Module provides a global function ‘all’ to fetch all records of all models which are registered with ‘register(model_name)’ before and their comments.

The submodule ‘Commentable’ is injected to ActiveResource and provides the method ‘is_commentable’ which injects the InstanceModule-methods of this module
